a. <code>

The <code> element is used to define a piece of computer code.

For example, in HTML, you can use <p> to define a paragraph.

b. <kbd>

The <kbd> element is used to define keyboard input.

Press Ctrl + C to copy the selected text.

c. <var>

The <var> element is used to define the name of a variable in mathematical or programming contexts.

Let x be the variable representing the width of a rectangle.

d. <samp>

The <samp> element is used to define sample output from a computer program.

The output of the program is: Hello, World!
